- # 文件复制工具 (File Copier)

  一个简单而强大的文件复制/移动工具，支持命令行和图形界面操作。

  ## 功能特点

  - 支持文件复制和移动操作
  - 可按文件扩展名筛选
  - 支持按关键词包含/排除文件
  - 可选择是否保留原有文件夹结构
  - 详细的操作日志记录
  - 进度条显示处理进度
  - 支持命令行和图形界面两种操作方式

  ## 安装

  1. 克隆或下载本仓库
  2. 安装依赖：
     ```
     pip install -r requirements.txt
     ```

  ## 使用方法

  ### 图形界面模式

  1. 直接运行启动脚本：
     ```
     python start_ui.py
     ```
     
  2. 或者通过命令行参数启动图形界面：
     ```
     python file_copier.py -g
     ```
